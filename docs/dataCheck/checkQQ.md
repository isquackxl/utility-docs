> 判断数据是否符合 QQ 号规范

## 引入

```js
import { checkQQ } from "@quackxl/utility";
```
or
```js
import { checkQQ } from "@quackxl/utility/cores/checkQQ.js";
```

## 格式

```js
checkQQ(str);
```

## 参数

| 属性名   | 类型      | 描述             |
| -------- | --------- | ---------------- |
| `str`    | `String`  | 需要判断的字符串 |
| `return` | `Boolean` |                  |
