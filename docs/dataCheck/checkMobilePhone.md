> 判断数据是否符合移动电话号码规范

## 引入

```js
import { checkMobilePhone } from "@quackxl/utility";
```
or
```js
import { checkMobilePhone } from "@quackxl/utility/cores/checkMobilePhone.js";
```

## 格式

```js
checkMobilePhone(str);
```

## 参数

| 属性名   | 类型      | 描述             |
| -------- | --------- | ---------------- |
| `str`    | `String`  | 需要判断的字符串 |
| `return` | `Boolean` |                  |
