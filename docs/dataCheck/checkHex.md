> 判断数据是否符合 hex 颜色格式规范

## 引入

```js
import { checkHex } from "@quackxl/utility";
```
or
```js
import { checkHex } from "@quackxl/utility/cores/checkHex.js";
```

## 格式

```js
checkHex(color);
```

## 参数

| 属性名   | 类型      | 描述             |
| -------- | --------- | ---------------- |
| `color`  | `String`  | 需要判断的字符串 |
| `return` | `Boolean` |                  |
