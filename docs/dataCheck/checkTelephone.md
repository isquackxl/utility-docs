> 判断数据是否符合固定电话号码规范

## 引入

```js
import { checkTelephone } from "@quackxl/utility";
```
or
```js
import { checkTelephone } from "@quackxl/utility/cores/checkTelephone.js";
```

## 格式

```js
checkTelephone(str);
```

## 参数

| 属性名   | 类型      | 描述             |
| -------- | --------- | ---------------- |
| `str`    | `String`  | 需要判断的字符串 |
| `return` | `Boolean` |                  |
