> 获取数据的类型

## 引入

```js
import { getType } from "@quackxl/utility";
```
or
```js
import { getType } from "@quackxl/utility/cores/getType.js";
```

## 格式

```js
getType(value);
```

## 参数

| 属性名   | 类型     | 描述                                     |
| -------- | -------- | ---------------------------------------- |
| `vaue`   | `Any`    | 需要获取数据类型的数据                   |
| `return` | `String` | 返回值。返回具体的类型字符串，如`String` |
