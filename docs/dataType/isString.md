> 判断数据是否为`String`类型

## 引入

```js
import { isString } from "@quackxl/utility";
```
or
```js
import { isString } from "@quackxl/utility/cores/isString.js";
```

## 格式

```js
isString(value);
```

## 参数

| 属性名   | 类型      | 描述           |
| -------- | --------- | -------------- |
| `value`  | `Any`     | 需要判断的数据 |
| `return` | `Boolean` |                |
