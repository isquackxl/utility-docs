> 判断数据是否为 `BigInt`类型

## 引入

```js
import { isBigInt } from "@quackxl/utility";
```
or
```js
import { isBigInt } from "@quackxl/utility/cores/isBigInt.js";
```

## 格式

```js
isBigInt(value);
```

**参数：**

| 属性名   | 类型      | 描述           |
| -------- | --------- | -------------- |
| `value`  | `Any`     | 需要判断的数据 |
| `return` | `Boolean` |                |
