> 判断数据是否为`Number`类型

## 引入

```js
import { isNumber } from "@quackxl/utility";
```
or
```js
import { isNumber } from "@quackxl/utility/cores/isNumber.js";
```

## 格式

```js
isNumber(value);
```

## 参数

| 属性名   | 类型      | 描述           |
| -------- | --------- | -------------- |
| `value`  | `Any`     | 需要判断的数据 |
| `return` | `Boolean` |                |
